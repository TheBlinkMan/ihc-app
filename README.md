IHC class project

####How to use in linux:
```
cd ihc-app
python3 -m venv venv
pip3 install -r requirements.txt
source venv/bin/activate
FLASK_APP=webapp.py flask run
```
####How to use in Windows :
```
cd ihc-app
python3 -m venv venv
pip3 install -r requirements.txt
venv/bin/activate.bat
set FLASK_APP=webapp.py
flask run
```
